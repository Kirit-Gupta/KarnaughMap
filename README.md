# KarnaughMap
This is my Karnaugh Map program. In the program you can input a Karnaugh Map with upto 4 inputs or an expression. The program will draw the loops on a Karnaugh Map as well as outputing the simplified expression. 

To run the program you must have python installed as well as having all scripts.
The program requires the following modules; Kivy, KivyMD and all other default Python libraries.
